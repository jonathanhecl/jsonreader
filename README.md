# jsonreader
JSON Simple Reader

## Requirements
* A JSON array

## Returned struct
- Headers []string
- Rows    map[int]map[string]string

## Installation
`go get github.com/jonathanhecl/jsonreader`

## Example
```go
import (
    "github.com/jonathanhecl/jsonreader"
)

func main() {
     // ...
}

```
